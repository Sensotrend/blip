# How to run blip

## Dependencies

You need node and npm.

    npm install express
    npm install ejs-locals
    npm install less-middleware
    npm install facebook-node-sdk
    
## Running it

    > node app.js
    Blip server listening on port 8081

Now open http://localhost:8081 and you should see the login screen.
